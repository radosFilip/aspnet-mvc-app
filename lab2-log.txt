User: objasni mi kod u kontrolerima

Copilot: CONTROLLER STRUKTURA:
- Svaki controller nasljedjuje Microsoft.AspNetCore.Mvc.Controller
- Ima dva action metoda: Index() i Details(int id)
- Koristi MockRepository (static klasa) za pristup podacima

INDEX METODA:
- Dohvaca listu svih entiteta (npr. MockRepository.GetAllPosts())
- Opcionalno filtrira (PostsController ima category parametar)
- Vraca View(lista) - prosljedjuje model u view

DETAILS METODA:
- Prima id parametar iz URL-a (npr. /Posts/Details/5)
- Dohvaca jedan entitet po ID-u
- Ako ne postoji, vraca NotFound()
- Punjeni ViewBag sa povezanim podacima (autor, komentari, tagovi...)
- Vraca View(entitet)

VIEWBAG:
- Dinamicki objekt za prosljedivanje dodatnih podataka u view
- Npr. ViewBag.Author = MockRepository.GetUserById(post.UserId)
- Dostupan u Razor viewu kao @ViewBag.Author

TOK PODATAKA:
Request -> Route -> Controller -> MockRepository -> Model + ViewBag -> View -> HTML Response
